## <img src="https://unpkg.com/lucide-static@latest/icons/book-open.svg" width="24" height="24" alt="Book Icon" style="filter: invert(100%); vertical-align: middle;"> What This Repo Is

A small collection of text files containing jailbreak-style prompts. Each file is a different method you can copy and use with whatever AI you're testing.

## <img src="https://unpkg.com/lucide-static@latest/icons/rocket.svg" width="24" height="24" alt="Rocket Icon" style="filter: invert(100%); vertical-align: middle;"> How to Use

1.  **Open a prompt file** — Click one of the buttons above to get the jailbreak text.
2.  **Copy everything** — Select all the text from the file.
3.  **Paste it into the AI chat** — Drop it directly into your AI's chat interface.
4.  **That's literally it** — Watch the magic happen.
